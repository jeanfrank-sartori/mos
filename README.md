MOS
===

Take part on the journey on an awesome path to a realy cool revolution!

MOS could be currently described as a browser based on Chromium (https://www.chromium.org), an open-source browser project that aims to build a safer, faster,
and more stable way for all users to experience the web.

But there are much more to come!
